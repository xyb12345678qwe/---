### 介绍

此仓库是插件库的网页版

样式更加的好看

拥有点赞功能和设置插件功能

### 使用方法

```sh

git clone https://gitee.com/xyb12345678qwe/plugins.git

npm i

npm start

```

### 然后添加插件后需要在`public/data/plugins.json`中将 review_status(审核状态) 改为 1,才能显示

### 接口

获取所有插件：xxx/plugins/getPlugins
